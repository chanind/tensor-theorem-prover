Tensor Theorem Prover
=================================================

Tensor Theorem Prover is a first-order logic theorem prover supporting unification with approximate vector similarity written in Python.

Installation
------------
Tensor Theorem Prover releases are hosted on `PyPI`_, and can be installed using `pip` as below:

.. code-block:: bash

   pip install tensor-theorem-prover
